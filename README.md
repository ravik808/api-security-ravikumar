# API Security Project

## Description
This project demonstrates secure API usage by removing hardcoded API keys, handling rate limiting, and protecting user privacy when using the OpenWeatherMap API.

## Answers

**1. What are the real-world consequences of exposing an API key on GitHub?**  
Exposing an API key can allow attackers to misuse the service, leading to unauthorized API calls, increased costs, service disruption, or account suspension. It can also result in data breaches and damage the company’s reputation.

**2. Why does your company's privacy policy prohibit logging city names?**  
City names can be considered personal data when combined with other information. Logging such data violates privacy principles like data minimization under regulations such as GDPR and may expose sensitive user location information.
